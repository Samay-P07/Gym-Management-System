# FITCORE - Gym Management System

A premium, modern Django-based web application designed to manage gym operations seamlessly. Built with high-end aesthetic UI designs and secure backend integrations, this system offers complete user membership tracking, dynamic fitness packages, responsive booking layouts, and a dedicated rebranded admin portal.

## 🚀 Key Features
- **Client Authentication**: Secure Member Sign Up, Login, and Password Management with email tracking.
- **Dynamic Pricing Plans**: View, interact with, and purchase various subscription tiers.
- **Payments & Booking**: Live tracking of user bookings, registration requests, and payment history securely.
- **Comprehensive Admin Dashboard**: Centralized hub to track all bookings, user queries, package categories, and dynamically generate administrative reports.
- **Premium UI / UX**: Crafted with specialized Glassmorphism design aesthetics, active flexbox layouts, SVG Favicons, and modern responsive CSS variables.

## 💻 Tech Stack
- **Backend Architecture:** Python, Django (Object-Relational Mapping & Auth)
- **Frontend Infrastructure:** HTML5, Advanced Vanilla CSS, Bootstrap 4
- **Database:** SQLite3
- **Icons & Tracking:** FontAwesome 6, Chart.js

## ⚙️ How To Run Locally

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Samay-P07/Gym-Management-System.git
   cd Gym-Management-System
   ```
2. Select your Python Environment (Virtual Env recommended).
3. Install the primary dependencies (Django).
4. Apply the database migrations:
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
5. Create a Superuser account to access the Admin Dashboard:
   ```bash
   python manage.py createsuperuser
   ```
6. Spin up the development server:
   ```bash
   python manage.py runserver
   ```
7. Open your browser and navigate to `http://127.0.0.1:8000/`.
